Domaine: Gestion d'une bibliothèque 
===================================

===> Cette application est composée de 2 projets:
	
	 EJB Project: biblioServer + Dynamic Web Project: biblioClient

Dans ce projet EJB (biblioServer), il faut utiliser les JAR suivants:
	- antlr-2.7.7.jar
	- c3p0-0.9.1.jar
	- commons-collections-3.2.1.jar
	- dom4j-1.6.1.jar
	- hibernate3.jar
	- hibernate-jpa-2.0-api-1.0.1.Final.jar
	- javassist-3.15.0-GA.jar
	- jta.jar
	- ojdbc6.jar
	- slf4j-api-1.6.1.jar
	
===> Ce projet Fonctionne avec le projet Web: biblioClient
